Hover over any area on the tablature and click. Select multiple cells by clicking and dragging the mouse. Hold ctrl and click any additional cells.

Type any number from 0 to 24.

Key:

  • 0-1: Enter a number into selected cells
  • Delete/Backspace: Clear input from all selected cells
  • Ctrl+Left Click: Select/unselect a cell while other cells are selected

This is a test prototype. Tabs cannot be saved.
